背景

CSS 背景属性用于定义元素的背景效果。背景可以是颜色、图像、渐变等。

盒模型

CSS 盒模型本质上是一个包围每个 HTML 元素的盒子。它包括：内容、内边距、边框和外边距。

盒模型示例

内容 (Content): 显示文本和图像的区域

内边距 (Padding): 内容周围的空白区域

边框 (Border): 内边距和内容周围的边框

外边距 (Margin): 边框外的空白区域
